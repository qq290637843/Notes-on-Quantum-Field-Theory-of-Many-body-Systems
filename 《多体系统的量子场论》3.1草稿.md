# 《多体系统的量子场论》3.1草稿

注意，$[a(\pmb x_1),a^\dagger(\pmb x_2)]=\delta(\pmb x_1-\pmb x_2)$
而根据两种定义方式有：$[a_{\pmb k_1},a_{\pmb k_2}^\dagger]=\delta_{\pmb k_1,\pmb k_2}$，$[a_{\pmb k_1},a_{\pmb k_2}^\dagger]=(2\pi)^d\delta(\pmb k_1-\pmb k_2)$

$$H=\int \frac{d^d\pmb k}{(2\pi)^d}\ \frac{\pmb k^2}{2m}a_{\pmb k}^\dagger a_{\pmb k}\\=\int\frac{d^d\pmb k}{(2\pi)^d}d^d{\pmb x}_1d^d{\pmb x}_2\ \frac{\pmb k^2}{2m}a^\dagger(\pmb x_1)a(\pmb x_2)\exp[i\pmb k\pmb x_1-i\pmb k\pmb x_2]\\=-\int d^d{\pmb x}_1d^d{\pmb x}_2\frac{1}{2m}\pmb\nabla\cdot\pmb\nabla\delta(\pmb x_1-\pmb x_2)a^\dagger(\pmb x_1)a(\pmb x_2)\\=-\int d^d\pmb x_1d^d\pmb x_2\frac1{2m}\delta(\pmb x_1-\pmb x_2)a^\dagger(\pmb x_1)\pmb\nabla\cdot\pmb\nabla a(\pmb x_2)\\=\int d^d\pmb x\ a^\dagger(\pmb x)\left(-\frac1{2m}\pmb\nabla\cdot\pmb\nabla\right)a(\pmb x)$$

$$iG(\pmb k,t)=i\int d^d\pmb x\ G(\pmb x,t)\exp[-i\pmb k\pmb x]\\=\int d^d\pmb x\ \langle a(\pmb x)\exp[-itH]a^\dagger(\pmb 0)\rangle\exp[-i\pmb k\pmb x]\\=\int d^d\pmb x\frac{d^d\pmb k_1}{(2\pi)^d}\frac{d^d\pmb k_2}{(2\pi)^d}\langle a_{\pmb k_1}\exp[-itH]a^\dagger_{\pmb k_2}\rangle\exp[i\pmb k_1\pmb x-i\pmb k\pmb x]\\=\int \frac{d^d{\pmb k_2}}{(2\pi)^d}\langle a_{\pmb k}\exp[-itH]a_{\pmb k_2}^\dagger\rangle\\=\langle a_{\pmb k}\exp[-it H]a_{\pmb k}^\dagger\rangle$$

问题 3.1.2.
1.
首先$$|\Phi_0\rangle=\frac{1}{\sqrt{N!(2\pi)^N\delta(0)^N}}(a_0^\dagger)^N|0\rangle$$

于是当$t>0$时：
$$iG(x,t)=\int\frac{dk_1}{2\pi}\frac{dk_2}{2\pi}\frac{1}{N!(2\pi)^N\delta(0)^N}\langle0|a_0^N\exp[iHt]a_{k_1}\exp[-iHt]a_{k_2}^\dagger(a_0^\dagger)^N|0\rangle\exp[ik_1x]\\=\int\frac{dk_1}{2\pi}\frac{dk_2}{2\pi}\frac1{N!(2\pi)^N\delta(0)^N}\langle0|a_0^Na_{k_1}\exp[-iHt]a_{k_2}^\dagger(a_0^\dagger)^N|0\rangle\exp[ik_1x]\\=\int\frac{dk_1}{2\pi}\frac{dk_2}{2\pi}\frac{1}{N!(2\pi)^N\delta(0)^N}\langle0|a_0^Na_{k_1}a_{k_2}^\dagger(a_0^\dagger)^N|0\rangle\exp[ik_1x-ik_2^2t/2m]$$

现在考虑wick定理所诱导的所有有效缩并，于是得
$$iG(x,t)=\int\frac{dk_1}{2\pi}\frac{dk_2}{2\pi}\frac{1}{N!(2\pi)^N\delta(0)^N}\left(N!(2\pi)^{N+1}\delta(k_1-k_2)\delta(0)^N\\+NN!(2\pi)^{N+1}\delta(k_1)\delta(k_2)\delta(0)^{N-1}\right)\exp[ik_1x-ik_2^2t/2m]\\=\int\frac{dk_1}{2\pi}\exp[ik_1x-ik_1^2t/2m]+N(2\pi)^{-1}\delta(0)^{-1}\\=\sqrt{-im/2\pi t}\exp[imx^2/2t]+N(2\pi)^{-1}\delta(0)^{-1}$$

而$t<0$时：
$$iG(x,t)=\int\frac{dk_1}{2\pi}\frac{dk_2}{2\pi}\frac1{N!(2\pi)^N\delta(0)^N}\langle0|a_0^Na_{k_2}^\dagger\exp[iHt]a_{k_1}\exp[-iHt](a_0^\dagger)^N|0\rangle\exp[ik_1x]\\=\int\frac{dk_1}{2\pi}\frac{dk_2}{2\pi}\frac1{N!(2\pi)^N\delta(0)^N}\langle0|a_0^Na_{k_2}^\dagger a_{k_1}(a_0^\dagger)^N|0\rangle\exp[ik_1x-ik_1^2t/2m]\\=\int\frac{dk_1}{2\pi}\frac{dk_2}{2\pi}\frac1{N!(2\pi)^N\delta(0)^N}NN!(2\pi)^{N+1}\delta(k_1)\delta(k_2)\delta(0)^{N-1}\exp[ik_1x-ik_1^2t/2m]\\=N(2\pi)^{-1}\delta(0)^{-1}$$

从而$iG(x,0^+)-iG(x,0^-)=\lim_{t\to0^+}\sqrt{-im/2\pi t}\exp[imx^2/2t]$，而注意
$$\sqrt{-im/2\pi t}\int dx\ \exp[imx^2/2t-ikx]=\exp[-ik^2t/2m]$$

从而$\lim_{t\to 0^+}\sqrt{-im/2\pi t}\exp[imx^2/2t]=\delta(x)$

注意$2\pi\delta(0)$也就是$\mathcal V$，从而非对角长程序就是$N/\mathcal V=n_0$
