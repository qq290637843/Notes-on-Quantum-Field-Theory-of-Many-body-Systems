# 《多体系统的量子场论》7.3.3草稿

## 问题 7.3.3.
1.携带单位电荷$-e$：
根据(7.3.23)
$$Q_{\pmb l}=-e\pmb l^TK^{-1}\pmb q$$

于是
$$Q_{\pmb l_e}=-e\pmb L^TKK^{-1}\pmb q=-e\pmb L^T\pmb q=-e$$

2.具有费米统计：
根据(7.3.23)
$$\theta_{\pmb l}=\pi\pmb l^TK^{-1}\pmb l$$

于是
$$\theta_{\pmb l_e}=\pi\pmb L^TKK^{-1}K\pmb L=\pi\pmb L^TK\pmb L=\pi\sum_{I,J}L_IK_{IJ}L_J\\=\pi\sum_IK_{II}L_I^2+2\pi\sum_{I<J}K_{IJ}L_IL_J$$

其中显然只有$K_{11}L_1^2$的奇偶性有讨论价值，注意$L_1=1$，证毕

3.绕任意任意准粒子一圈的相位是$2\pi$的整数倍：
根据互统计相位公式
$$\theta_{\pmb l^{(1)},\pmb l^{(2)}}=2\pi\pmb l^{(1)T}K^{-1}\pmb l^{(2)}$$

于是
$$\theta_{\pmb l,\pmb l_e}=2\pi\pmb l^TK^{-1}K\pmb L=2\pi\pmb l^{T}\pmb L$$

证毕
4.完备性：
现在要证明，如果一个激发满足前三个条件，则其一定能写成$K\pmb L,\pmb q^T\pmb L=1,\pmb L\in\mathbb Z^n$。
设未知激发为$\pmb l_x$，根据条件3可知
$$\pmb l^TK^{-1}\pmb l_x,\forall\pmb l\in\mathbb Z^n$$

故
$$\pmb L=K^{-1}\pmb l_x\in\mathbb Z^n$$

根据条件1可知
$$\pmb q^TK^{-1}K\pmb L=1$$

故
$$\pmb q^T\pmb L=1$$
