# 《多体系统的量子场论》3.7.5草稿

## 问题3.7.4.

$$L=\frac{1}{8\pi e^2}(c^{-1}\pmb E^2-c\pmb B^2)\\=\frac{1}{8\pi e^2c}(\partial_0A_i)^2+\frac{1}{8\pi e^2c}(\partial_iA_0)^2-\frac{1}{4\pi e^2c}\partial_iA_0\partial_0A_i\\-\frac{c}{8\pi e^2}\epsilon_{ijk}\partial_jA_k\epsilon_{iuv}\partial_uA_v$$

考虑取规范$A_0=0$，于是
$$L=\frac{1}{8\pi e^2c}(\partial_0A_i)^2-\frac{c}{8\pi e^2}\epsilon_{ijk}\partial_jA_k\epsilon_{iuv}\partial_uA_v$$

所以经典运动方程为：
$$\partial_0^2A_k-c^2\epsilon_{ijk}\epsilon_{iuv}\partial_j\partial_uA_v=0$$

傅里叶后就是
$$-\omega^2A_k+c^2\epsilon_{ijk}\epsilon_{iuv}k_jk_uA_v=0$$

而
$$\epsilon_{ijk}\epsilon_{iuv}k_jk_uA_v=\epsilon_{ijk}\epsilon_{iuv}k_jk_u(\hat k_vA^\parallel+(\hat n_1)_vA_1^\perp+(\hat n_2)_vA_2^\perp)\\=\epsilon_{ijk}k_j|\pmb k|((\hat n_2)_iA_1^\perp-(\hat n_1)_iA_2^\perp)=k_ik_i(\hat n_a)_kA_a^\perp$$

故
$$-\omega^2A^\parallel=0\\-\omega^2A_a^\perp+c^2k_ik_iA_a^\perp=0$$

所以$\omega=c|\pmb k|$
