# 《多体系统的量子场论》3.3.7草稿

其实(3.3.19)后根据$[H,W]=0$得出$|\Phi_0\rangle$是$W$本征态这个行为有点循环论证的嫌疑，因为这件事情依赖于有限系统基态无简并，但是问题是这一节本来就是要说明有限系统基态无简并。

注意，(3.3.21)中，$\theta_0$的共轭动量$p$，其实就是$-N$，因为：
$$\delta\rho=-\frac{\partial_t\theta}{V_0},p=\frac{\mathcal V\partial_t\theta}{V_0}-\frac{\mu\mathcal V}{V_0},\mu=\rho_0V_0$$

于是
$$p=-(\rho_0+\delta\rho)\mathcal V=-N$$

从而(3.3.22)中的$n$就是$N$

注意，$H$的表达式有笔误，应为
$$H=\frac{p^2}{2M}+\mu p$$

## 问题 3.3.4.
首先，根据(3.3.18)可知$$v_s=\sqrt{\frac{V_0\rho_0}{m}}$$故可知
$$V_0=\frac{v_s^2m^2}{\rho_{\rm mass}}$$

注意哈密顿量为：
$$H=\frac{p^2}{2\mathcal V/V_0}+\mu p$$，从而实际上构成了一个“质量”为$\mathcal V/V_0$的自由粒子在$\theta$空间中的运动。

注意$\Delta\theta_0\Delta p\sim1/2$，从而
$$\Delta\partial_t\theta_0=\frac{\Delta p}{\mathcal V/V_0}\sim\frac{1/2}{\Delta\theta_0\mathcal V/V_0}=\frac{1/2}{\mathcal V\pi/10V_0}$$

从而$$\Delta\theta_0(t)\sim\frac{5V_0t}{\mathcal V\pi}$$

而我们需要$\Delta\theta_0(t)$达到$2\pi$

故$$t\sim\frac{2\pi^2\mathcal V}{5V_0}=\frac{2\pi^2\mathcal V\rho_{\rm mass}}{5v_s^2m^2}\\=\frac{2\pi^2({\rm cm^3})\times0.14513{\rm g/cm^3}}{5(238.2{\rm m/s})^2(6.646\times10^{-27}{\rm kg})^2}\times1.055\times10^{-34}{\rm J\cdot s}\\=2.412\times10^{10}{\rm\ s}=764.8{\rm\ years}$$
