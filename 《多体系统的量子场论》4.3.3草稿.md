# 《多体系统的量子场论》4.3.3草稿

## 问题 4.3.5.
首先
$$\Pi^{00}(\omega,k)=\int\frac{dq}{2\pi}\frac{n_F(\xi_{q-k/2})-n_F(\xi_{q+k/2})}{\omega-(\xi_{q+k/2}-\xi_{q-k/2})+i0^+}\\=\int\frac{dq}{2\pi}\frac{n_F(\xi_{q-k/2})-n_F(\xi_{q+k/2})}{\omega-qk/m+i0^+}\\=\int\frac{dq}{2\pi}\frac{n_F(\xi_{q-k/2})}{\omega-qk/m+i0^+}-\int\frac{dq}{2\pi}\frac{n_F(\xi_{q+k/2})}{\omega-qk/m+i0^+}\\=\int\frac{dq}{2\pi}\frac{n_F(\xi_q)}{\omega-qk/m-k^2/2m+i0^+}-\int\frac{dq}{2\pi}\frac{n_F(\xi_q)}{\omega-qk/m+k^2/2m+i0^+}\\=-\frac{m}{2\pi k}\int_{-k_F}^{+k_F}dq\ \left(\frac{1}{q-\omega m/k+k/2-i0^+}-\frac{1}{q-\omega m/k-k/2-i0^+}\right)\\=-\frac{m}{2\pi k}\log\frac{\omega-v_Fk-k^2/2m+i0^+}{\omega+v_Fk-k^2/2m+i0^+}+\frac{m}{2\pi k}\log\frac{\omega-v_Fk+k^2/2m+i0^+}{\omega+v_Fk+k^2/2m+i0^+}\\=\frac{m}{2\pi k}\log\frac{\omega^2-(v_Fk-k^2/2m)^2+i{\rm sgn}(\omega)0^+}{\omega^2-(v_Fk+k^2/2m)^2+i{\rm sgn}(\omega)0^+}\\=\begin{cases}\frac{m}{2\pi k}\log\frac{\omega^2-(v_Fk-k^2/2m)^2}{\omega^2-(v_Fk+k^2/2m)^2},|\omega|>|v_Fk+k^2/2m|{\rm\ or\ }|\omega|<|v_Fk-k^2/2m|\\\frac{m}{2\pi k}\log\frac{\omega^2-(v_Fk-k^2/2m)^2}{(v_Fk+k^2/2m)^2-\omega^2}-{\rm sgn}(\omega)\frac{mi}{2k},|v_Fk-k^2/2m|<|\omega|<|v_Fk+k^2/2m|\end{cases}$$

接着，由于$$\partial_t\rho+\partial_xj=0$$所以
$$\Pi^{01}=\frac{\omega}{k}\Pi^{00},\Pi^{10}=\frac{\omega}{k}\Pi^{00},\Pi^{11}=\frac{\omega^2}{k^2}\Pi^{00}$$
