# 《多体系统的量子场论》3.7.1草稿

## 问题3.7.1.
1.
$$L\to i\frac12\sum_i\left(\varphi_i^* (\partial_0+i\partial_0f_i+iA_0(i)-i\partial_0f_i)\varphi_i\\-\varphi_i(\partial_0-i\partial_0f_i-iA_0(i)+i\partial_0f_i)\varphi_i^*\right)\\+\sum_{\langle ij\rangle}\left(t_{ij}\varphi_j^*\varphi_i\exp[if_i-if_j-ia_{ij}+if_j-if_i]\\+t_{ij}\varphi_i^*\varphi_j\exp[if_j-if_i+ia_{ij}+if_i-if_j]\right)=L$$

2.
$$\frac{d}{dt}\frac{\partial L}{\partial(\partial_0\varphi_i^*)}=\frac{\partial L}{\partial\varphi_i^*}$$

注意一个问题，原表达式中的$\sum_{\langle ij\rangle}$是对每条边记一次，且反向边不记，故

$$-i\frac12\partial_0\varphi_i=i\frac12\partial_0\varphi_i-\frac12A_0(i)\varphi_i-\frac12A_0(i)\varphi_i\\+\sum_{j\in{\rm adj}(i)}t_{ij}\varphi_j\exp[ia_{ij}]$$

即
$$\partial_0\varphi_i=-iA_0(i)\varphi_i+i\sum_{j\in{\rm adj}(i)}t_{ij}\varphi_j\exp[a_{ij}]$$

3.
注意题干中需要调整为$\partial_0(\varphi_i^*\varphi_i)$。
于是
$$\partial_0(\varphi_i^*\varphi_i)=-iA_0(i)\varphi_i\varphi_i^*+i\sum_{j\in{\rm adj}(i)}t_{ij}\varphi_j\varphi_i^*\exp[a_{ij}]\\+iA_0(i)\varphi_i^*\varphi_i-i\sum_{j\in{\rm adj}(i)}t_{ij}\varphi_j^*\varphi_i\exp[-ia_{ij}]\\=i\sum_{j\in{\rm adj}(i)}t_{ij}(\varphi_i^*\varphi_j\exp[ia_{ij}]-\varphi_j^*\varphi_i\exp[-ia_{ij}])$$

从而
$$J_{ij}=it_{ij}(\varphi_j^*\varphi_i\exp[-ia_{ij}]-\varphi_i^*\varphi_j\exp[ia_{ij}])$$

4.
$$-\frac{\partial L}{\partial a_{ij}}=it_{ij}\varphi_j^*\varphi_i\exp[-ia_{ij}]-it_{ij}\varphi_i^*\varphi_j\exp[ia_{ij}]=J_{ij}$$
